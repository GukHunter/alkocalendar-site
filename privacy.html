2. Какие данные мы обрабатываем

В зависимости от используемых функций приложения мы можем обрабатывать:

  • Данные аккаунта: email, идентификатор пользователя, никнейм, аватар, пол.
  • Данные авторизации: технические токены входа, данные OAuth-провайдера (например, Google).
  • Данные профиля и использования: записи о напитках, календарные отметки, прогресс, статистика, достижения, данные по кланам/чату.
  • Технические данные: push-токен устройства, тип устройства, версия приложения, служебные журналы ошибок.
  • Коммуникационные данные: сообщения в чате с ботом (для формирования ответов и улучшения стабильности работы чата).

3. Цели обработки данных

  • регистрация и вход в приложение;
  • предоставление функций приложения (календарь, статистика, кланы, уведомления, чат);
  • синхронизация данных между устройствами пользователя;
  • обеспечение безопасности, предотвращение злоупотреблений и отладка ошибок;
  • отправка сервисных уведомлений (включая push, если включены пользователем).

4. Правовые основания обработки

  • исполнение пользовательского соглашения (предоставление функциональности сервиса);
  • согласие пользователя (например, на push-уведомления, обработку отдельных категорий данных);
  • законный интерес оператора (безопасность сервиса, предотвращение мошенничества, поддержка работы приложения).

5. Хранение и срок обработки

Данные хранятся в течение срока использования сервиса пользователем или до удаления аккаунта, если иной срок не требуется по закону. Технические журналы могут храниться ограниченное время для диагностики и безопасности.

6. Передача данных третьим лицам

Для работы приложения могут использоваться внешние инфраструктурные провайдеры:

  • Supabase — аутентификация, база данных, серверные функции;
  • Google OAuth (при входе через Google) — авторизация;
  • Push-сервисы (например, FCM) — доставка уведомлений;
  • AI/API-провайдеры для чат-функций и генерации изображений (только в объёме, необходимом для ответа/генерации).

Мы не продаём персональные данные пользователей. Передача осуществляется только в пределах, необходимых для работы функций приложения.

7. Международная передача данных

Часть данных может обрабатываться на серверах, расположенных за пределами страны пользователя. Используя приложение, пользователь соглашается с такой передачей в рамках работы сервиса.

8. Безопасность данных

Мы применяем разумные технические и организационные меры защиты: разграничение доступа, токены авторизации, защищённые соединения и мониторинг ошибок. Однако ни один способ хранения/передачи данных в интернете не может гарантировать абсолютную безопасность.

9. Push-уведомления

Приложение может запрашивать разрешение на push-уведомления. Пользователь может включить/отключить их в настройках устройства и внутри приложения.

10. Чат и AI-функции

Сообщения пользователя в чате могут обрабатываться для формирования ответов. Не рекомендуется передавать в чат чувствительные персональные данные (например, паспортные данные, банковские реквизиты, медицинские документы).

11. Права пользователя

Пользователь вправе:

  • запросить доступ к своим данным;
  • запросить исправление или удаление данных;
  • отозвать согласие на отдельные виды обработки (где применимо);
  • обратиться по контактам оператора по вопросам обработки данных.

12. Удаление аккаунта и данных

Пользователь может запросить удаление аккаунта и связанных данных через поддержку: [УКАЖИТЕ EMAIL ПОДДЕРЖКИ]. После обработки запроса данные удаляются или обезличиваются, кроме случаев, когда хранение требуется по закону.

13. Данные несовершеннолетних

Сервис предназначен для совершеннолетних пользователей. Если вы считаете, что несовершеннолетний передал нам данные без законных оснований, свяжитесь с нами для удаления информации.

14. Медицинский дисклеймер

Приложение носит информационный характер. Расчёты и рекомендации не являются медицинским заключением и не заменяют консультацию врача.
